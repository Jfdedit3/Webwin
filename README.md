# Webwin Gallery

Webwin Gallery is a Windows desktop photo gallery built with Electron.

## Features

- Open a local folder containing images
- Recursive image scan in subfolders
- Photo grid with thumbnails
- Large preview area
- File details panel
- Search by filename
- Sort by name, date, or size
- Previous and next navigation
- Slideshow mode
- Windows installer build with electron-builder

## Run locally

```bash
npm install
npm run start
```

## Build for Windows

```bash
npm install
npm run build
```

The installer is generated in the `release` folder.
