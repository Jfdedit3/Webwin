# Call of War Android Web App

Android wrapper app focused on displaying only `https://callofwar.com` inside a WebView.

## Features

- Opens only `callofwar.com` and related subdomains
- JavaScript enabled
- DOM storage enabled
- In-app navigation
- External non-allowed links opened outside the app
- Edge-to-edge fullscreen style

## Project type

Native Android app using Kotlin + WebView.

## Open in Android Studio

Open the repository root as an Android project, then build and run on a device or emulator.

## Main files

- `settings.gradle.kts`
- `build.gradle.kts`
- `app/build.gradle.kts`
